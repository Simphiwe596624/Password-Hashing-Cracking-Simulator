# Password-Hashing-Cracking-Simulator
This project demonstrates password security by simulating hashing and cracking using the SHA-256 algorithm.It was first built in Python with Tkinter for dictionary based cracking, then converted to HTML,CSS, and JavaScript for better usability and visuals.The project is educational, helps beginners understand weak passwords and cybersecurity risks.
